<SectionCard

  <DataTable
    headers={["技術", "フルネーム", "役割"]}

    ]}
  />

  <Callout type="info" title="VEXの重要性">
    SBOMだけでは「大量の脆弱性リスト」に圧倒されます。VEXを使うことで「この脆弱性は含まれるが、設定により悪用不可である」ことを伝え、不必要な調査を減らせます。
  </Callout>
</SectionCard>

  <AttackFlow label="攻撃の手口">

    <strong>Step 3:</strong> `axios@1.14.1` などを公開。`plain-crypto-js@4.2.1` を依存関係に追加し `postinstall` フックで自動実行。<br/><br/>

# 緊急対応: 安全なバージョンへダウングレードとキャッシュクリア

npm cache clean --force
rm -rf node_modules package-lock.json
npm ci`}
  </Terminal>
