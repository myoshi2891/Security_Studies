<Callout type="info" title="AIは「信頼できないジュニアエンジニア」">
  AI生成コードは必ず人間がレビュー。ブランチ保護ルールで1名以上の承認を必須に設定しましょう。
</Callout>

```yaml title=".github/copilot-content-exclusion.yml"
# Copilot Content Exclusion設定
# 機密ファイルをAI推論コンテキストから除外
content_exclusion:
  paths:
    - "**/.env*"
    - "**/*.pem"
    - "**/*.key"
    - "**/internal-config.*"
    - "**/secrets/**"
    - "**/.aws/**"
```
